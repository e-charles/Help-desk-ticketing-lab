# Help-desk-ticketing-lab

Uses JIRA to simulate tickets, categories, priority levels, and escalation paths. 
1. Ticket categories like account access, hardware, software, and network.
2. Set priority for urgent vs routine issues.
3. Triage workflow with status changes such as (New, In Progress, Waiting on User, and Resolved).
4. SLA targets such as response within 1 hour and resolution within 1 business day.
5. A sample knowledge base article linked to a common issue.
